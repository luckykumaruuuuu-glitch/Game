SECRET KEY SESSION LOGIC FIX
================================
CURRENT LOGIC IS WRONG
================================
Right now, when Secret Key is activated in one match, it remains activated in future matches.
This is incorrect.
================================
CORRECT BEHAVIOR
================================
Secret Key must be tied to a specific match session.
================================
EXAMPLE
================================
Match A
↓
User enters Secret Key
↓
Secret Key Activated
↓
Monster Icon Visible
↓
Secret Features Unlocked
================================
IF USER TEMPORARILY EXITS
================================
If the user:
Closes the game screen
Leaves the room temporarily
Gets disconnected
Rejoins the SAME match
Then:
✅ Secret Key must remain activated
✅ Monster Icon must remain visible
✅ Secret Features must still work
Because it is still the SAME match.
================================
WHEN TO DEACTIVATE
================================
ONLY deactivate Secret Key when:
Match A is completely finished
Room A is permanently closed
A brand new match is created
A new room is started
================================
EXAMPLE
================================
Match A
↓
Secret Key Activated
↓
User exits temporarily
↓
User rejoins Match A
↓
Secret Key STILL ACTIVE
(Correct)
================================
BUT
================================
Match A Ends
↓
Create Match B
↓
Secret Key MUST be OFF
↓
Monster Icon Hidden
↓
Secret Features Locked
↓
User must enter Secret Key again
(Correct)
================================
IMPORTANT RULE
================================
Secret Key should persist for the CURRENT MATCH ONLY.
Secret Key should NOT persist across NEW MATCHES.
Same Match = Keep Activation
New Match = Reset Activation
================================
TECHNICAL REQUIREMENT
================================
Store Secret Key state using Match ID / Room ID.
Secret Key activation should belong to that specific match session.
When a new Match ID is created:
Clear Secret Key state
Hide Monster Icon
Lock Secret Features
User must activate Secret Key again for the new match. :::

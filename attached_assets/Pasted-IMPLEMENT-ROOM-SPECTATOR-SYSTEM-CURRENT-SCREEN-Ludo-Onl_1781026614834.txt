================================

BUTTON PLACEMENT

================================

The Room button must ONLY appear on the Online Friend main screen.

Do NOT show this button:

- Inside game screen
- Inside match board
- Inside room lobby after joining
- Inside any other screen

Only show it on the Online Friend home screen next to Active and Invite.

================================

ROOM BUTTON FUNCTION

================================

When user taps Room:

Open popup dialog.

Title:

Enter Room ID

Input:

Room ID Text Field

Buttons:

- Watch Match
- Cancel

================================

ROOM VALIDATION

================================

When user enters Room ID:

1. Check if room exists.

2. Check if room is currently active.

If room does not exist:

Show:

Room not found

If room has ended:

Show:

Match has ended

If room is active:

Allow entry as Spectator.

================================

SPECTATOR MODE

================================

Spectator is NOT a player.

Spectator must NEVER:

- Roll dice
- Move tokens
- Join turn system
- Take any game action
- Affect game state

Spectator can ONLY watch.

================================

REALTIME MATCH VIEW

================================

After entering valid Room ID:

Open live board.

Spectator must see in realtime:

- Dice rolls
- Dice numbers
- Token movements
- Current turn
- Player names
- Player status
- Exit/Kick status
- Chat messages (if chat exists)
- Match progress

Everything must update instantly.

================================

FIREBASE STRUCTURE

================================

Keep players and spectators separate.

Example:

rooms
└── roomId
├── players
├── spectators
├── gameState
└── chat

Spectators must never be added to players collection.

================================

2. Friend Chat

EMOJI REACTION BEHAVIOR

================================

When an emoji is sent:

Show ONLY emoji.

Do NOT show message bubble.

Animation:

1. Emoji appears near top area
2. Slight pop animation
3. Smooth floating movement
4. Move downward slightly
5. Stay visible for 1-2 seconds
6. Fade out automatically

Emoji should feel like a gaming reaction.

Make emoji larger than text messages.

================================

TEXT MESSAGE BEHAVIOR

================================

When a text message is sent:

Show:

Player Name + Message

Example:

Lucky: Nice Move!

Rahul: 😄

================================

MESSAGE ANIMATION

================================

Messages appear from upper area.

Move downward slowly.

Do NOT reach the keyboard area.

Before reaching the bottom controls:

- Fade out
- Disappear automatically

Display time:

Around 8-10 seconds

================================

REALTIME RULE

================================

Messages and emojis must be visible to:

✔ All players

✔ All spectators

Realtime synchronization required.

================================

ANTI-SPAM

1 reaction every 1 second.
